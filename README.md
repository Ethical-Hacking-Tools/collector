# Install 
```
git clone https://github.com/galihap76/collector.git
cd collector 
pip install -r requirements.txt
```
For termux :
```
pkg update && pkg upgrade
pkg install python3
pkg install git
git clone https://github.com/galihap76/collector.git
cd collector
pip install -r requirements.txt
```

# Usage
```
python3 main.py -h
```
